О моем проекте
Привет! Я создал этот репозиторий, чтобы показать, как я учусь программировать на Java. Я не хотел просто зубрить теорию, поэтому написал небольшую программу для проверки данных.
Как всё устроено
Я разделил код на две части, чтобы в нем было легко разобраться. В одном файле лежат сами правила (например, как программа проверяет, что пароль не слишком короткий или что номер телефона начинается с +998). В другом файле я просто запускаю эти проверки, чтобы сразу увидеть результат в консоли — работает всё правильно или есть ошибки.
