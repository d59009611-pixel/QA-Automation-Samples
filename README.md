# QA Automation Samples

Проект по валидации данных пользователя для стажировки в **Yandex Pro**.

**Что реализовано:**
- Проверка номера телефона Узбекистана с очисткой от лишних символов.
- Валидация имени (только буквы, длина от 2 символов).
- Проверка безопасности пароля.

**Технологии:**
- Java 11
- Регулярные выражения (Regex) для проверки строк.
