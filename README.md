# 🚀 QA Automation Samples (Java)

Привет! Я создал этот проект, чтобы показать свои навыки на практике перед подачей на стажировку в **Yandex Pro**. 

### 🎯 Что это за проект?
Здесь я написал простую систему проверки (валидации) данных пользователя. Это то, с чем постоянно сталкивается тестировщик: проверка корректности ввода номера телефона, пароля или имени.

### 💻 Что внутри:
*   **UserValidator.java**: Логика проверок (номер телефона Узбекистана, длина пароля).
*   **Main.java**: Код для запуска и демонстрации того, что всё работает правильно.
*   **Проверка граничных условий**: Я учел, что данные могут быть пустыми (`null`), чтобы программа не падала с ошибкой.
